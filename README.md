# Fake News Detector (NLP + Machine Learning)

A beginner-friendly but resume-worthy project that detects fake news using
text preprocessing, NLP, and machine learning.

## Project Structure
- `data/` — raw & processed datasets
- `src/` — all python scripts (loading, preprocessing, training, prediction)
- `notebooks/` — experiments & analysis

## Steps Completed
- Step 1: Folder structure created
- Step 2: Dataset loaded & combined
- Step 3: Text cleaned (stopwords removed, case normalized)

## Next Steps
- Step 4: Train ML model
- Step 5: Evaluate performance
- Step 6: Build a web app

## Dataset
Download Fake.csv and True.csv from:
https://www.kaggle.com/competitions/fake-news/data
